# PokeScanner Pro

ポケモンカードの総合ツール。AI鑑定・用語集・カード図鑑・資産管理・在庫マップの5タブ構成。

## 機能

- **スキャン**: Claude APIによるカード画像の自動識別・鑑定・PSA価格表示
- **用語集**: 40+の専門用語（コンディション・鑑定・サーチ対策等）
- **図鑑**: カード画像検索・シリーズ一覧・レアリティ表・高額カード図鑑
- **資産**: ポートフォリオ管理・価格チャート・損益追跡
- **マップ**: カードショップ・在庫情報の位置登録

## 使い方

`pokescanner.html` をブラウザで開くだけで動作します（サーバー不要）。
